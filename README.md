---

## 🔧 **Installation & Setup**

### Step 1: Get API Key
1. Go to [Google AI Studio](https://makersuite.google.com/app/apikey)
2. Click "Get API Key"
3. Create new API key
4. Copy the key (keep it secret!)

### Step 2: Open in Colab
[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/Harshitha-44S/corporate_assistant/blob/main/corporate_assistant.ipynb)

### Step 3: Run the Code
1. Paste your API key when prompted
2. Enter company details
3. Select document type
4. Provide requirements
5. Get your generated document!
